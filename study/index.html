# 小学生托管

玉环新睿托管是一所专业从事学生中托、晚托（作业辅导）、周末托、暑寒假托管，在发展中不断创新和探索，创造独特的学生托管教育模式，在同类托管信社会中树立了良好的形象，并且在学生行为习惯矫正及提升学生成绩方面取得可喜的成绩，得到在托学生家长的一致好评！完成做到让家长放心、省心。

我们愿和家长一起努力，共同帮助孩子进步，孩子收获到的不仅是满意的成绩，更是良好的行为习惯和优秀的个人品德。

托管内容：日托、晚托、午托、周末托、暑寒假托管

我们的托管理念

第一、我们尊重孩子的个性成长，注重孩子全面发展

第二、我们把当今的应试教育和素质教育完美结合到我们的托管服务中。

第三、我们关注孩子的成绩，更注重孩子的素质培养。

我们的目标：办玉环最好的托管

我们的宗旨：用我们的爱心、责任心、让家长省心、放心，让孩子开心。

# 高效阅读

高效阅读法使扩大学生阅读量具有可能性。 高效阅读法呈现了用计时阅读训练激发学生的紧迫感和效率感的科学性。 高效阅读法体现探索阅读规律、实现训练的规律化。高效阅读是语文教改的科研成果，是阅读教学上的一个具有突破性的变革，它对于整个语文教改都是很有启发意义的。

# 研学活动

为拓展学生学习的空间， 丰富学生的学习经历和生活体验， 让学生能在研学的过程中陶治情操、增长见识、体验自然和人文环境、提高学习兴趣，全面提升综合素质。

# 专注力训练

孩子有足够的专注力，上课才能专心听讲，做事认真。 那我们有什么全脑开发专注力训练的方法呢？让新睿全脑教育为你揭秘。 宝宝的学习能力其实是和专注力有很大的关系的，所以父母从小对孩子的专注力的锻炼是很关键的。
