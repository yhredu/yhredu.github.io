# 教学环境

玉环新睿托管是一所专业从事学生中托、晚托（作业辅导）、周末托、暑寒假托管，在发展中不断创新和探索，创造独特的学生托管教育模式，在同类托管信社会中树立了良好的形象，并且在学生行为习惯矫正及提升学生成绩方面取得可喜的成绩，得到在托学生家长的一致好评！完成做到让家长放心、省心。 我们愿和家长一起努力，共同帮助孩子进步，孩子收获到的不仅是满意的成绩，更是良好的行为习惯和优秀的个人品德。

托管内容：日托、晚托、午托、周末托、暑寒假托管

我们的托管理念: 第一、我们尊重孩子的个性成长，注重孩子全面发展 第二、我们把当今的应试教育和素质教育完美结合到我们的托管服务中。 第三、我们关注孩子的成绩，更注重孩子的素质培养。

我们的目标：办玉环最好的托管

我们的宗旨：用我们的爱心、责任心、让家长省心、放心，让孩子开心。
